## ⚙️ Setting Up the Agent

After you install Python, you need to prepare the folder where you extracted the project:

1.  Open the folder named `build-ai-agents-free`.
2.  Click the folder path bar at the top of the window. Type `cmd` and press the Enter key. A black window will appear.
3.  Type `pip install -r requirements.txt` into this black window and press Enter. This command downloads the tools your agent needs to function. Wait for the process to finish.

## 🚀 Running Your First Agent

Once the setup finishes, you are ready to start.

1.  Keep the same black window open.
2.  Type `python main.py` and press Enter.
3.  The agent will initialize. It will ask you for a task. Type your goal clearly, such as "Research the weather in London and save the result to a text file."
4.  The agent will begin to think and perform steps. It uses open-source models to process your request. Watch the black window to see the agent explain its progress.

## 🛠️ Frequently Asked Questions

**Does this software cost money?**
No. All components included in this project are open-source and free to use. You do not need a paid API key.

**Can I run this offline?**
The initial setup requires an internet connection to download the required files. Once the files reside on your computer, the agent works based on the local models you choose.

**What do I do if I get an error?**
Most errors happen during the installation of Python. Ensure you checked the "Add to PATH" box. If you see an error about a missing file, run the `pip install` command again to ensure all tools downloaded correctly.

**Is this safe?**
The software runs on your computer. It does not send your personal files to a cloud server unless you tell it to interact with a specific website. You control what the agent does at all times.

**How do I stop the agent?**
In the black window, press the `Ctrl` key and the `C` key at the same time to stop the program safely.
